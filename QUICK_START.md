## Quick start if required modules are installed

### Make sure to install nodejs & npm in your system
https://medium.com/@imvinojanv/how-to-install-node-and-npm-on-your-local-machine-45fc0f2438a2

### Terminal 1
```bash
cd server
npm install
npm start
```

### Terminal 2
```bash
cd client
npm install
npm start
```
